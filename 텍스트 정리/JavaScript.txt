<자바스크립트 오류>
- undefined : 값이 할당되지 않은 변수
  => 값이 저장 되지 않았는데 출력
- null : 주소가 들어있지 않음(객체) 
  => 참조형
- NaN : 값이 들어있지 않음(number, 숫자) 
  => 기본형
- querySelector("#id")-> CSS 선택자랑 같음
#1
1. 이름과 휴대폰번호를 입력 받아서 출력
2. 게시판 내용은 input 태그에서 입력을 받아서
   서버로 전송 후, 전송된 내용 확인
3. 두 수를 입력 받아서
   더하기, 빼기, 곱하기, 나누기 함수를 각각 정의 후, 
   함수 호출하여 처리결과 출력
4. 버튼 3개를 만든 후, 카운터 구현
   값을 감소시키는 버튼 : -1씩 감소
   값을 증가시키는 버튼 : +1씩 증가
   값을 초기화시키는 버튼 : 0으로 초기화
5. 네이버 버튼을 누르면 네이버 사이트로 이동
   다음 버튼을 누르면 다음 사이트로 이동
   구글 버튼을 누르면 구글 사이트로 이동
