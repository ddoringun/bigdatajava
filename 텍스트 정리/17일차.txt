1. 오버로딩이란?
- 메소드를 재정의 하는 것
2. 오버라이딩이란?
- 다형성 : 하나의 이름으로 다양한 형태를 구현하는 자바의 특징
3. 오버로딩과 오버라이딩의 차이는?
- 오버라이딩은 상속을 받아야 사용이 가능하고 오버로딩은 x
- 오버로딩은 값을 수정할 수 x / 오버라이딩 값 수정 가능
4. 단일상속이란?
- 하나의 자식 클래스에는 하나의 부모클래스만 상속받는 것.
5. 상속이란?
- 기존클래스를 재사용 하는 것
6. 다차원상속이란?
- 단계별로 부모클래스를 접근하는 것
7. 수퍼클래스란?
- 부모클래스
8. 서브클래스란?
- 자식클래스
9. 부모, 자식클래스를 나타내는 키워드는 각각?
- 부모 : super  /  자식 : this
10. 부모, 자식 중 누가 더 일반적인 특징을 가지는가?
- 부모클래스 (Generic)
